# 💰 Gerenciador de Gastos Pessoais

Sistema simples em Python para controle financeiro, desenvolvido como desafio técnico.

## 🚀 Como Executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o programa: `streamlit run app.py`
3. Link do app: https://bootcampii-desafio-inicial-j6gtfxfhatgwlzgekkpqpu.streamlit.app/

## 🧪 Como Rodar os Testes
Execute: `python -m pytest`

## 🛠️ Tecnologias
- Python 3.10+
- Pytest (Testes Unitários)
- GitHub Actions (CI)
